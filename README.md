# univie-gpa-calc
 
Scrape your grades from u:space and calculate average grade.

## Installing requirements
Windows:
```
py -3 -m pip install -r requirements.txt
```

macOS/Linux:
```
python3 -m pip install -r requirements.txt
```

## Running the script:
Windows:
```
py -3 scraper.py
```

macOS/Linux:
```
python3 scraper.py
```
